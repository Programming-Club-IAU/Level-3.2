## Run Locally

Clone the project

```bash
  git clone https://github.com/FarisAI24/SocialMediaAppUI.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  flutter pub get
```

Start the server

```bash
  flutter run
```
